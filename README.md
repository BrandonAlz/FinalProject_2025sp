# Final Quest

Final Quest is a game made by a group of students: Brandon de Oliveira, Kyle Hasenkamp, Avikumar Patel, Ana P, Carlos S, Sear Sultani, Nicholas Torres and Sarah Watters. Together we created a 2D RPG/Bullet hell game with (limited) multiplayer support leveraging AWS Lambda and Amplify integrations to the cloud.

The game can be played at: https://staging.d1x42rvkvcnnsq.amplifyapp.com/
